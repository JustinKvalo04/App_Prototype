# Cross-Platform Shopping Filters App

## Overview

The Cross-Platform Shopping Filters App is a UX design project focused on simplifying online shopping across multiple e-commerce platforms.

Many shoppers repeatedly apply the same filters (price range, size, color, brand, availability) across different websites. This project explores a solution that allows users to define their preferences once and automatically apply them across multiple shopping platforms.

## Problem

Online shoppers often spend significant time re-entering the same filters when comparing products across different websites.

This repetitive process creates friction and reduces the efficiency of product discovery.

## Solution

I designed a prototype that allows users to:

* Create and save shopping preferences
* Select preferred filters once
* Apply those filters across multiple shopping platforms
* Streamline product discovery and comparison
